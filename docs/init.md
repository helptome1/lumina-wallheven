我想要设计一款桌面端壁纸应用，主要风格是毛玻璃 + 半透明。类macOS Ventura。
页面左右布局
1. 左侧是菜单及分类（在线壁纸，热门推荐，动漫精选，人物精选，我的收藏，下载中心，关于）
2. 右侧顶部上面是图片尺度选择及搜索框（选择按钮：SFW，Sketchy，NSFW，热门列表下拉框：（最热，最新，随机，热门列表，最多收藏），分辨率下拉框：（1080p+,2K+, 4K+, 8K+）,搜索框）
3. 右侧主体：图片瀑布流，一行最大5个图片，响应式布局，可随屏幕大小调整每行图片数量。
4. 鼠标悬浮在图片上时，显示出图片收藏数和图片尺寸，还有收藏按钮。
5. 点击图片，查看图片详情，通过弹窗放大图片。
6. 整体背景是当前图片第一张的图片，通过毛玻璃虚化。点击左侧菜单后，后面背景进行更换
整体氛围：柔和的光影、细腻的阴影、轻微的发光边缘。配色以冷色调（淡蓝、淡紫）为主，部分点缀色为柔和的橙色。分辨率16:10，4K画质，Behance风格。`

字体颜色：rbg(32,33，36)
背景颜色：rbg(241,243,244)
