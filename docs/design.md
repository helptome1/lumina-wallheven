# 三、真正决定“高级感”的关键

很多人以为：

> 高级感 = 炫酷

其实不是。

真正高级的是：

---

## 1. 留白

不要把界面塞满。

高端 UI 最大特点：

- 大面积呼吸空间
- 内容聚焦
- 控件极少

像：

- Arc
- Notion
- Linear

---

## 2. 动效“轻”

不要：

- 到处弹窗
- 到处飞入
- 到处旋转

高级 UI 的动效：

- 很慢
- 很柔
- 很克制

比如：

- 200ms opacity
- 轻微缩放
- 缓慢 blur
- 柔和阴影

---

## 3. 色彩控制

推荐：

### 主色：

```text
#0B0F1A
#121826
#1A2333
```

### 点缀：

```text
#7C5CFF
#4DA3FF
#00D1B2
```

### 禁止：

- 高饱和红
- 纯黑纯白
- 彩虹渐变

---

## 4. 字体决定气质

推荐：

### 英文：

- SF Pro
- Inter
- Geist
- Satoshi

### 中文：

- HarmonyOS Sans
- MiSans
- 思源黑体

### 重点：

- 字重轻
- 行高大
- 不要粗黑字体
